# Welcome to the Scarlet Knowledge Base

Use this knowledge base to move faster through certification with Scarlet.

## What You'll Use It For

You'll use the content in two ways:

- **Understand requirements:** What regulators expect for EU MDR and ISO 13485 certifications.
- **Understand the process:** How Scarlet runs audits, assessments, and post-certification activities.

- **Applying for ISO 13485 certification?** Start with the ISO 13485 quality management page
- **Need help?** Use the in-app chat to reach us
