# Release Notes

Change log for the Scarlet app.

## March 17th, 2026

### UK MDR Application and Transfer Forms Available

You can now apply for UK MDR certificates with a webform rather than a spreadsheet. This mirrors the experience offered for ISO 13485 and EU MDR.

### Updated Coverage Checker Design

We've improved the coverage checker design to make it easier to use:

- Reduced visual clutter by removing repeated information
- No need to expand accordions to view details
- Improved readability with colour coding and citation modals

## February 24th, 2026

### Customer Facing Audit Experience

We have released a new live audit experience for customers. Customers now have an audit page where they can see the details of their audit (eg date and time, meeting link) and can easily track questions as they come through from auditors throughout the day. Questions are clearly marked when completed and are accessible for reference later if required.

### Coverage Reports Can Be Downloaded

You can now export your coverage report to save it for later or share with team members who do not have access to Scarlet.

### Finding Response Import Allows Files to Be Grouped into Folders

You can now import a folder of files to be used to respond to queries, alongside just individual files. This enables a use case where you want to create a folder of impacted files for each query and then import all folders without having to manually write into the spreadsheet which file relates to which query.

## February 17th, 2026

### Finding Responses Present in Finding Exports

Now, if you have started drafting finding responses in the app before exporting the findings to CSV, those responses are available in the export.

### Knowledge Base User Experience Improvements

We added some improved UI styling to reduce the cognitive load of larger articles:

- Embedded training videos within the KB instead of using external links to LearnWorlds
- Images now zoom in/out
- Supports different colour call out blocks styles to denote key best practices (green), common pitfalls (orange), critical mistakes (red) and useful hints (blue)
- Added expandable blocks for deep dive reading content

We also simplified the sidebar navigation.

## February 10th, 2026

### New Homepage

We have developed a simple landing page which shows you the different categories of requests you can expect from Scarlet. Each is also represented in the sidebar, and shows the number of open items in that category.

### Transfer Webform

If you want to transfer certificates you have from other notified bodies to Scarlet, you can now enter the required information into an easy to use webform. Talk to your account manager if this is something you're interested in doing.

### Enhanced Search

The searchbar now shows results from across requests and findings as well as knowledge base article results.

## February 3rd, 2026

### Finding IDs Are Now Visible on Findings

You can now see the associated ID with each finding in both the sidebar and the main findings view. This finding is also available in the findings export and is used when importing findings in bulk to Scarlet.

### Users Can Be Given Access to Just Scarlet's Knowledge Base

Users can now be given access to Scarlet's knowledge base, without having access to the rest of the Scarlet application.

## January 26th, 2026

### Bulk Finding Responses Import: Part 2

You can now import files in response to findings. The flow is:

1. Select to export the findings
2. Fill out the Response and Filenames columns. The Filenames column is optional to fill in -- some findings may not require a file in response
3. Select to import
4. Upload all response files
5. Upload your filled out sheet
6. Confirm you are happy with the contents of the upload, resolve any errors and link additional files using the file dropdown if desired
7. Click Submit
8. Your findings have now been responded to!

### Priority Metadata Added to Findings

Assessors may mark some findings as 'High priority' which you should address first. These findings now appear at the top of the assessment sidebar with a 'High Priority' label.

## January 19th, 2026

### Email Notifications

When we publish new requests, applications or findings, you will now receive an email notification letting you know. The emails will come from Scarlet App Notifications and will contain a link that you can use to navigate to the relevant part of the Scarlet application.

### New Technical Assessment Finding Type: Request for Information

We have added a third finding type alongside Query and Observation: Request for information. These findings will typically come at the start of an assessment, and are where we feel information is missing or unclear, rather than in breach of requirements. Please prioritise these findings when you see them come through as they will be blocking further progress in our assessments.

## January 12th, 2026

### Bulk Finding Responses Import: Part 1

You can now import responses to findings in bulk. You can do this by:

1. Select to export the findings
2. Fill out the Response column (Filenames is optional)
3. Select to import
4. Upload your filled out sheet
5. Confirm you are happy with the contents of the upload and resolve any errors
6. Click Submit

Each finding will now have the response filled out automatically. You will still need to go in and add in the relevant files for each finding and submit them. We will be adding bulk file upload to this feature by the end of the month, which will utilise the filenames column.

If you do fill out the filenames column now, the filename(s) will be appended to your query response text.

### Findings Are Now Sorted into Devices

Findings are sorted under which device they relate to in the sidebar, so you can better understand the status of your device assessments. The device name is also shown on the main findings page and as a new column in the findings export.

### Improvements to the Application Form Experience

We have improved the error-handling in the application form so the page will automatically scroll to an error, and the errors are highlighted in the table of contents on the right-hand side. After you successfully submit an application, we have also improved the read-only version of the form, making it clearer that it is no longer editable.

## January 5th, 2026

### Coverage Checks Are Now Generated Automatically

Coverage check results now generate automatically in less than 15 minutes! Simply add any files, click 'Request Coverage' to receive your coverage report. Once you have the report, you can click 'Delete' in the top right to clear the report and run a report on new or additional files.

## December 22nd, 2025

### Query Responses Can Be Withdrawn

You can now withdraw query responses after submitting them, as long as the assessment team has not already resolved the query. This is intended for if you submit in error and need to urgently change something.

## December 8th, 2025

### Web Application Forms

We have introduced application forms into the application, so you can fill out all of your application details, upload technical files and QMS documentation easily in one place. Fields in the form autosave so you and your team can build your application over time. When ready, someone in your team can sign and submit the application to share it with Scarlet. You will see any feedback on the form visible directly in the app and will be able to adjust your submission as required.

### Intercom Chat Has Been Replaced with Chatway

We have replaced our previous chat vendor with Chatway, in order to provide a smoother customer experience. You can livechat with our customer team through the black widget in the bottom right.

### Interactive Folders Introduced

You have always been able to drop folders into Scarlet, alongside individual files. This update means that the folders exist as individual entities, rather than metadata on a file. You can expand and collapse the folder using a toggle, and can select to delete or download files from each folder individually.

## December 1st, 2025

### Zip Files Are Automatically Unzipped

You can continue to upload zip files to Scarlet and now see the contents automatically unzipped. This makes it easier to manage your files individually.

## November 24th, 2025

### Knowledge Base Search Launched

You can search through the knowledge base to find relevant articles. The search is available on all screens in the application so that you can search when the need arises.

## November 17th, 2025

### Evidence Files in Findings Can Be Previewed

When you receive queries or observations from our technical assessors, they will reference files that you have submitted as evidence. You can select these files and the file previewer to look through the file in question.

### Drag and Drop File Upload Improved

After you have selected a request, you can drag and drop a file anywhere on the page to upload it to the request.

## November 10th, 2025

### Knowledge Base Article Sidebar State Is Preserved

When selecting an article, the sidebar used to scroll back up to the top of the page. This issue has been fixed.

### Queries Design Updates

Small updates were made to how queries are represented in the Assessment tab to make them easier to view and respond to.

## October 27th, 2025

### New Sidebar

We have launched a new sidebar to make it easier for you to navigate the application. Within the sidebar, you can find links to the requests UI, the coverage tool and the knowledge base, as well as the link that allows you to logout of Scarlet. The sidebar expands when you hover over it, to show the available options.

## October 20th, 2025

### Coverage Report Is Now Separate to Requests

You can now engage with the coverage report without needing to upload files to requests. Navigate to the coverage report tab to upload files and generate a coverage report.

### Filename Restrictions Relaxed

We have removed restrictions on what filenames can be uploaded, so you can now include special characters like slashes and hyphens in filenames.

### Sign-in Expiry Time Extended

You now only need to re-sign in to Scarlet once every 30 days. If you are inactive for 7 days, you will be asked to sign-in again.

## October 13th, 2025

### Copy Prompt Feature Added to the Knowledge Base

When viewing an article about submission documents (eg Clinical Evaluation Plan), you can copy a prompt of the article and paste it into an LLM of your choice, along with your document, to get advice on how you could improve your documents in line with the recommendations from the article. This is available via simply clicking a button at the top of the article. If you do not see the button at the top of an article, it means that it isn't relevant to a specific submission document.

## October 6th, 2025

### Application Queries Can Now Be Viewed in the App

After you respond to an application request, you can see any queries that have been raised directly in the requests. You can then make the appropriate changes to your files and submit a response again.

## September 29th, 2025

### Knowledge Base Articles Are Now Accessible Directly Through the App

Knowledge base articles can be accessed via https://scarlet.cc/knowledge. There is also a link available in the header to switch between the knowledge base and your requests from Scarlet.

## September 15th, 2025

### Understand What Stage Your Scarlet Queries Are In

You can now see the different stages of the certification journey your queries are related to -- Pre-submission, Submission, Audit and Assessment. Each stage is represented by a tab at the top of the Scarlet application, and shows a count of open queries within the stage.

## September 1st, 2025

### View and Respond to Your Technical Assessment Queries in the Web

Technical assessment queries can be viewed and responded to directly in the app.

## August 25th, 2025

### Requests Have Become Centralised in the Application

You can now manage files directly in requests, rather than in the Scarlet drive. Responding to requests has also become easier -- you no longer need to create snapshots, instead just clicking 'Respond' directly.

### File Preview Functionality Added

You can now preview the files you have uploaded to the Scarlet app using your browser preview functionality.

## August 18th, 2025

### User Details Dropdown Added

You can now see the user you are logged in as in the top right of the Scarlet app topbar. Selecting this will provide you with a link to logout of Scarlet, and to view our API documentation.

## August 11th, 2025

### Folders Available in the File Table

You can now upload folders to the Scarlet drive, rather than just files. The folder structure will be shown in the Scarlet drive to help you communicate the structure of your submissions to Scarlet assessors.

## August 4th, 2025

### Coverage Tool Launched

We have built a coverage tool to help you understand how well your intended submission aligns with our guidance and best practices.

It provides directional feedback by identifying where expected concepts may appear in your documents. This can help you track progress and better understand the content referenced. Note: It does not evaluate quality, completeness, or regulatory sufficiency.

## July 28th, 2025

### New Request Filtering

You can now filter your requests depending on their status (Open, Responded, Closed and All Requests). There is a visual indicator now of how many requests are Open and require your attention.

### Updated Login Page

The login page flow has been updated with improved UX and new visuals. You can login using Google, Github, Microsoft, or one time OTP codes. Once you select a login method, we will remember your preferred method and offer that to you as default from now on.

### Uploaded By Field

The file table now displays who in your company uploaded a file to the Scarlet drive.

## July 21st, 2025

### Action Bar

You can now download or delete multiple files at once via an action bar. Simply select individual files using the checkboxes next to them. You can also select all files in the drive using the checkbox at the top and take the same actions on them.

## July 14th, 2025

### File Search

You can now search through the files you have uploaded to the Scarlet Drive via a new search bar at the top right of the page, above the drive. This will search through file names but not file contents.
